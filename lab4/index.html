Simple Computer Game Pseudocode List:

  1. Title Screen
  2. Accept user input to start
  3. Await user input for players (players: 2)
  4. Accept user input (players: 2)
  5. Game Start
  6. Players start with 2 hands, 1 finger each
  7. Accept player 1 move
  8. Validate player 1 move
    9. <1i>Tap 1 of the opponent’s hand to add 1 finger (point) to opponent’s hand
  9. If finger # is greater than 1
  10. Then add finger # amount to opponent’s hand
  11. If player has even total # fingers, player can tap hands together to move half # fingers to other hand
  12. Player input: If finger # is 3 on one hand (3 & 1)
  13. Then add additional finger to single finger hand (2 & 2)
  14. Player input: If finger # is 4 on one hand (4 & “dead” hand)
  15. Then split into 2 fingers on each hand (2 & 2)
  16. Evaluate possible moves from player 1
  17. Accept player 2 move
  18. Validate player 2 move
    19. <1i>Tap 1 of the opponent’s hand to add 1 finger (point) to opponent’s hand
  19. If finger # is greater than 1
  20. Then add finger # amount to opponent’s hand
  21. If player has even total # fingers, player can tap hands together to move half # fingers to other hand
  22. Player input: If finger # is 3 on one hand (3 & 1)
  23. Then add additional finger to single finger hand (2 & 2)
  24. Player input: If finger # is 4 on one hand (4 & “dead” hand)
  25. Then split into 2 fingers on each hand (2 & 2)
  26. Evaluate possible moves from player 2
  27. Test of win condition
  28. If a player’s hand reaches 5 finger count
  29. Then a player loses a hand
  30. If a player reaches 0 hands
  31. Then player loses and opponent wins
